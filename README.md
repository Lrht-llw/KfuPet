## 项目介绍

KfunPet（快芙桌宠）是一款基于 **WPF** 开发的桌面宠物应用，采用 **混合动作系统**，结合本地预设动画与 AI 实时驱动，在低成本、高性能的前提下，实现富有创意的角色交互表现。

### 核心特性

| 特性 | 说明 |
|------|------|
| 混合动作系统 | 预设动画保证流畅，AI 驱动实现创意即兴动作 |
| 伪骨骼系统 | 精细化控制 Head、Body、Arm、Leg 等部件的旋转与位移 |
| 智能调度器 | 预设优先策略，优先匹配本地动作库，无命中时才调用 AI 生成，大幅节省 tokens 消耗 |
| 动作沉淀 | AI 生成的优质动作可沉淀为预设，持续扩展动作库 |
| 情绪感知 | 根据当前情绪状态自动匹配并播放对应动作 |

### 动作调度策略

系统采用 **预设优先** 的调度策略，以最小化 AI 调用成本：

```
动作请求 → 查询预设动作库 → 命中? → 是 → 直接播放预设动画
                              ↓ 否
                    调用 AI 生成动作序列
                              ↓
                    播放 AI 生成的动画
                              ↓
                    (可选) 沉淀为预设
```

- **基础动作**（打招呼、眨眼、伸懒腰等）直接调用预设动作库，零 AI 成本
- **AI 提议动作** 先查预设库，命中则复用，未命中才让 AI 自主生成
- **动作沉淀**：AI 生成的优质动作可保存到预设库，避免重复消耗 tokens

## 未来构想

### 构想一：桌宠互联

打造跨设备、跨用户的桌宠社交网络。

- **设备互联**：当用户拥有两台及以上安装桌宠的设备时，可选择是否开启互联。开启后，不同设备间的桌宠可以互相"串门"，在设备间自由穿梭。
- **用户互联**：用户之间可以互相添加好友，好友的桌宠之间也能实现互动与拜访。
- **陌生人随机访问**：用户可选择开启"随机串门"功能，系统会随机匹配其他在线用户，让你有机会遇到陌生人的桌宠，带来意外的惊喜与互动。

### 构想二：ESP32-S3 二次元桌面时钟

基于 **ESP32-S3** 开发一款独立运行的二次元桌面时钟硬件。

- **基础功能**：保留传统桌面时钟的时间显示，支持多种时钟界面风格。
- **融入桌宠**：在时钟界面中嵌入桌宠模块，让时钟不再单调，桌宠可陪伴你的每一次查看。
- **设备互联**：二次元时钟可与 PC 桌宠及其他 ESP32-S3 时钟设备实现互联，共享桌宠互动体验。
- **通讯方式**：
  - **蓝牙 / WiFi**：近距离设备间直连，低延迟、高私密。
  - **互联网**：远距离跨网段通信，随时随地远程互联。

### 互联体系总览

| 设备 | PC 桌宠 | ESP32-S3 时钟 |
|------|--------|--------------|
| PC 桌宠 | ✅ 设备间互访 | ✅ 跨设备互联 |
| ESP32-S3 时钟 | ✅ 跨设备互联 | ✅ 设备间互访 |
| 陌生人随机访问 | ✅ | ✅ |
| 近距离（蓝牙/WiFi） | — | ✅ |
| 远距离（互联网） | ✅ | ✅ |
